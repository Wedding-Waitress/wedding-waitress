# Wedding Waitress 💍

Wedding Waitress is a modern web app designed to help couples and event planners organise weddings and events with ease.

## 🎉 Features

- Digital invitations & RSVP tracking
- Guest list management
- Table and seating planning
- QR code seating charts
- Photo & video sharing (coming soon)
- Event planning tools in one simple dashboard

## 💡 What makes it different?

Wedding Waitress simplifies event planning by combining multiple tools into one easy-to-use platform — no spreadsheets, no stress.

## 🌍 Who it's for

- Couples planning their wedding
- Event planners
- Anyone organising an event (birthdays, engagements, corporate events)
